
# To initialize the scripts
```
terraform init
```



# To plan
```
terraform plan
```


# Apply the changes
```
terraform apply
```
or

```
terraform apply -auto-approve
```



# To remove the instance 
```
terraform destroy
```


# To display the instances configuration
```
terraform show
```


# To validate the code block
```
terraform show
```


# To correct the line of code block
```
terraform show
```
